Sim. O GeradorXML utiliza a API oficial ADN (adn.nfse.gov.br) com autenticação mTLS, que requer o certificado digital A1 (.pfx) da empresa ou do escritório com procuração eletrônica. Sem o certificado A1 não é possível acessar os dados via API oficial.
O GeradorXML funciona em qualquer navegador, mas o processo de upload do certificado A1 (.pfx) é mais simples pelo computador do escritório, onde o arquivo normalmente está salvo. No celular, você pode criar sua conta e explorar o painel, mas para processar XMLs recomendamos acessar pelo desktop.
Para períodos curtos (1–3 meses) e volumes de até 200 notas, o processo leva entre 30 segundos e 2 minutos. Para volumes maiores (acima de 500 notas) ou períodos mais longos, pode levar até 5 minutos. O progresso é exibido em tempo real na tela.
